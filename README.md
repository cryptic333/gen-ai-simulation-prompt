# 🤖 Generative AI Copilot Toolkit

A Python-based interactive toolkit that simulates **Copilot-style AI workflows** for everyday knowledge tasks such as document summarisation, email drafting, and data interpretation.

Built as part of a hands-on exploration into **Generative AI adoption, prompt engineering, and workflow design**.

---

## 🚀 Features

- 📝 **Document Summarizer**  
  Summarises long text into bullet points, paragraphs, or executive summaries  

- ✉️ **Email Drafter**  
  Generates structured, professional emails based on purpose, tone, and key inputs  

- 📊 **Data Interpreter**  
  Explains raw data in plain English with insights and recommendations  

- 🧠 **Prompt Engineering Guide**  
  Interactive lessons demonstrating best practices for writing effective AI prompts  

- 🧪 **Prompt Tester**  
  Test and refine your own prompts with feedback  

- 📌 **KPIs Dashboard**  
  Simulated metrics to evaluate AI adoption (efficiency, quality, usage)

---

## 🛠️ Tech Stack

- **Python 3.11**
- **Streamlit** (Web UI)
- Modular architecture (separate functional components)

---

## ⚠️ Note on AI API Usage

Due to API quota limitations, this project is configured to run in **Mock Mode** by default.

👉 This allows:
- Full UI interaction  
- Workflow testing  
- Demonstration without requiring API keys or billing
